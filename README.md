<h1>ZeroTrace – College Project Frontend</h1>
<h2>Project Overview</h2>
<p>This repository contains the frontend code for ZeroTrace, a college project simulating a modern cybersecurity company website. The purpose is to demonstrate skills in web design, responsive layouts, and interactive features typical of a real-world SaaS/security provider landing page.

All content, product names, and company details are <i>fictional and created for academic presentation</i> only.</p>

<h2>Features</h2>
<ul type="square">
  <li>Multi-page structure: Home, About, Services, Products, Subscriptions, News & Events, Learn/Resources, Contact
</li>
<li>Modern, responsive UI</li>
<ul type="circle">
<li>Dark-themed styling for a tech/cyber look (styles.css)
</li>
  <li>Mobile-optimized layouts</li>
</ul>

<li>Animated & interactive elements</li>
<ul type="circle">
<li>Sticky header, hamburger menu for navigation
</li>
<li>Tabs for filtering services/resources</li>

<li>FAQ accordion, newsletter sign-up, contact form with validation</li>

<li>Scroll, fade-in, and counter animations</li>

<li>Pricing toggle for plans (monthly/annual)</li>
</ul>

<li>Component-based sections: Service cards, team bios, product comparisons, resource lists</li>

<li>Client-side form validation: All forms provide instant feedback using vanilla JS</li>
</ul>

  <h4>How It Works</h4>
  <ul>
    <li><strong>index.html</strong>: Main landing page, overview of solutions.</li>
    <li><strong>about.html</strong>: Company vision, leadership, achievements.</li>
    <li><strong>services.html</strong>: All core cybersecurity services listed and explained.</li>
    <li><strong>products.html</strong>: Product catalogue and feature comparison table.</li>
    <li><strong>subscription.html</strong>: Pricing plans, extra features, and FAQs with toggle and accordion interactivity.</li>
    <li><strong>news-events.html</strong>: Latest updates, news articles, and event announcements.</li>
    <li><strong>learn.html</strong>: Guides, webinar info, step-by-step resources for cybersecurity knowledge.</li>
    <li><strong>contact.html</strong>: Office locations, hours, interactive contact form, emergency contact details.</li>
    <li><strong>styles.css</strong>: All custom styles, color variables, and responsive design rules.</li>
    <li><strong>script.js</strong>: Responsible for all page interactivity (navigation, forms, tabs, animation).</li>
  </ul>

<p>You can open index.html directly in your browser, or use "Live Server" in VS Code for development.</p>


<h2>Tech Stack</h2>

<li>Vanilla JavaScript: All functionality and animations, no external libraries</li>

<li>No backend or database—just front-end simulation for this project</li>
</ul>
